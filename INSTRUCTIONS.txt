Instructions
------------
1. Download data from http://archive.ics.uci.edu/ml/machine-learning-databases/census-income-mld/census-income.data.gz

Model generations
-----------------
2. Open prototype.r in an R editor.
3. Change the first line to setwd("<directory of data>")
4. Run prototype in R
5. It should generate several models.

Reading data into pandas on Python
----------------------------------
6. Make sure readdata.py is in the same directory as data
7. Make sure pandas is installed for Python
8. Open ipython
9. run readdata.py
10. Should now be able to access the dataframe that is stored into "data"
